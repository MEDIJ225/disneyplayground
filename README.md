# dme
Edge Delivery Services project for dme.com

## Environments
- Preview: https://main--dme--twdc-minnie.aem.page/
- Live: https://main--dme--twdc-minnie.aem.live/

## Installation

```sh
npm i
```

## Linting

```sh
npm run lint
```

## Local development

- Install the [AEM CLI](https://github.com/adobe/helix-cli): `npm install -g @adobe/aem-cli`
- Start AEM Proxy: `aem up` (opens your browser at `http://localhost:3000`)
- Open the `dme` directory in your favorite IDE and start coding :)
